# RediRedi English Landing Page

This repository contains the English version of the RediRedi landing page, optimized for performance and streamlined for a focused user experience.

## Run Locally

**Prerequisites:** Node.js

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the app:
    ```bash
    npm run dev
    ```

## Project Structure

- `components/`: Modular React components.
- `utils/`: Utility functions (UTM handling, etc.).
- `public/`: Static assets.
- `vite.config.ts`: Vite configuration with optimized base path.

## Deployment

The project is configured to be deployed at `/sales-platform/`.
